Pocket Export RSS-XML/CSV/HTML
===========

Export your Pocket data using the [API](http://getpocket.com/developer/docs/overview) (formerly Read It Later) 

# Usage

Just clone the repo here and refer to the server.rb.

```sh
bundle install
ruby server.rb
```

or

```sh
docker-compose up
```

Then go to:

http://localhost:4567/
